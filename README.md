# CluckClock — Marketing site

Single-page site for **cluckclock.app**. Host on GitHub Pages.

## Hosting on GitHub Pages

1. Push this repo to GitHub.
2. **Settings → Pages** → Source: **Deploy from a branch**.
3. Branch: **main** (or your default), folder: **/ (root)**.
4. Save. The site will be at `https://<username>.github.io/<repo-name>/` or your custom domain.

## Before you publish

- Replace **`[APP_STORE_LINK]`** in `index.html` with your real App Store URL (both CTAs use it).
- Optionally add a **favicon** (app icon) and update **Support** / **Privacy policy** links in the footer.
